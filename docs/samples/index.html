    サンプル

    このページでは、Csv2Table.js の機能を利用した実践的なサンプルコードを紹介します。

    例 1: UIの日本語化（i18n オプション）

    この例では、基本的な CSV データを読み込み、カード型のレスポンシブなデザイン (デフォルト動作) を適用しています。 また、i18n オプションを使用してUIを日本語に設定しています。

    JavaScript:

    new Csv2Table({
    containerId: 'table1',
    csvUrl: 'sample1.csv',
    responsive: true, // デフォルト (画面幅が狭いとカード型になる)
    i18n: {
        searchPlaceholder: '検索...',
        filterColumns: '列の表示/非表示 ▼',
        noData: 'データが見つかりません',
        errorLoading: 'CSVデータの読み込みに失敗しました。'
    },
    columns: [
        { type: 'string' },  // 名前 
        { type: 'date' },    // 生誕日 
        { type: 'number' },  // 享年
        { type: 'url' },     // wiki
        { type: 'image' }    // 肖像
    ]
});
    CSV:

    sample1.csv
    名前,生誕日,享年,wiki,肖像
"織田信長",1534-07-03,49,https://ja.wikipedia.org/wiki/織田信長,nob.jpg
"豊臣秀吉",1537-03-27,62,https://ja.wikipedia.org/wiki/豊臣秀吉,hide.jpg
"徳川家康",1543-02-10,75,https://ja.wikipedia.org/wiki/徳川家康,yas.jpg

    Note

    彼らが生まれ年となった時代の西暦にはユリウス歴が使われていたが、ISO 8601に従うJavaScriptの日付処理ではユリウス歴は反映されず、仮想上のグレゴリオ暦換算の西暦日付となるため、ここでもその年月日で統一した。また、歴史人物の生誕日には諸説があり、そのうちの有力とされているものとした。

    例 2: 横スクロール ＆ カスタムCSS

    レスポンシブ処理を false に設定し、テーブルが画面幅を超えた場合は横スクロールバーを表示させます。 また、自動付与される CSS クラスを用いて、特定の列を右寄せ・赤色などに装飾する例です。

    CSS:

    /* 折り返し強制禁止 */
#table2 th,
#table2 td {
    white-space: nowrap;
}

/* ヘッダー全体を青に */
.csv2table-table2 .csv2table-thead-tr th { 
    background-color: #0056b3; 
    color: white; 
    border-color: #004494; 
}
.csv2table-table2 .csv2table-thead-tr th .csv2table-header-text::after {
    color: #fff;
    opacity: 0.8;
}

/* Populationの列（Index 2）を右揃え、赤太文字に */
.csv2table-table2 .csv2table-col-2 { 
    text-align: right; 
    font-weight: bold; 
    color: #d9534f; 
}

    JavaScript:

    new Csv2Table({
    containerId: 'table2',
    csvUrl: 'sample2.csv',
    responsive: false, // レスポンシブ横積みを無効にし、横スクロールを強制
    columns: [
        { type: 'number' },  // Rank
        { type: 'string' },  // Country
        { type: 'number' },  // Population
        { type: 'string' },  // Capital
        { type: 'url' },     // Info
        { type: 'image' }    // Flag
    ]
});

    例 3: 初期検索・列の非表示オプション、カスタム日付、コールバック

    initialSearch や hiddenColumns オプションを利用して、表が表示された瞬間にあらかじめフィルタリングがかかった状態を作るサンプルです。

    また、カスタムの日付フォーマットを指定して、CSV上に日本語で記述された M月D日 を日付フォーマットとして読み込み、日付でソート可能にしています。

    onRenderComplete のコールバックにて、外部スクリプトとの柔軟な連携が可能です。この例では、CSVのアイコン列内に holder.js のプレースホルダーを記述しておき、レンダリングによって表のDOMが動的に生成されたあとで、Holder.run を呼び出してアイコンを生成させています。DOMが構築されていることが前提のJavaScriptライブラリを使う場合に便利です。

    JavaScript:

    new Csv2Table({
    containerId: 'table3',
    csvUrl: 'sample3.csv',
    responsive: true, // Default
    initialSearch: 'Age:17', // 初期状態で「年齢列が17」のもので絞り込み
    hiddenColumns: ['URL'], // 列名「URL」を非表示
    columns: [
        { type: 'string' },  // Name
        { type: 'date', format: 'M月D日' },    // Birth Date (Custom Format) 
        { type: 'number' },  // Age
        { type: 'url' },     // URL
        { type: 'image' }    // Dummy Icon (holder.js)
    ],
    onRenderComplete: function (instance) {
        var myImages = document.querySelectorAll('#table3 .csv2table-img');
        Holder.run({
            images: myImages
        });
    }
});
